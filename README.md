> *Building the research → creation → publishing pipeline for Meta ads*

- 📚 **Ad Inspiration Library** — 1M+ ads indexed for trend research
- 🔍 **Live Search Engine** — real-time scraping of Meta ads with quality ratings, hook ID & transcriptions
- 📊 **Eve (Analytics Module)** — connects Meta accounts, categorizes creatives across 4 marketing funnels
- 🤖 **ADAM (SPY Agents)** — daily autonomous competitor scraping with automated performance analysis

### 💹 StockTax Pro *(Personal Project)*
> *Tax planning & optimization engine built for Indian traders*

`React` `TypeScript` `Tailwind` `Zustand` `SheetJS` `GPT-4o-mini`

- 🏦 Multi-broker support — AngelOne, Zerodha, Upstox with auto-format detection under one PAN
- 🌾 **Tax Loss Harvesting optimizer** — recommends minimum stocks to sell for maximum tax offset
- 🤖 GPT-4o-mini integration for personalized harvest recommendations & wash sale risk flagging
- 📐 **Tax Simulation engine** — what-if modeling, advance tax planning (Sec 234B/234C), scenario comparison
