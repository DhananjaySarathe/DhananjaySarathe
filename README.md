I'm a full-stack engineer who went from solving DSA problems in a dorm room to building AI products used at scale.
I care about one thing: **shipping things that actually work.**

Right now I'm a Founding SDE at [Quickads](https://quickads.ai), where I've built the product from scratch —
AI-powered ad research, real-time scraping engines, and autonomous competitor intel agents.

On the side I'm building **StockTax Pro** — a tax loss harvesting tool for Indian traders
because the existing options are terrible and I was tired of overpaying.

And I run **[ChilledBeer](https://www.youtube.com/@ChilledBeer)** on YouTube.
Real engineering. Real side projects. No polished highlights reel.
